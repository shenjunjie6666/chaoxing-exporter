# 🚀 学习通练习题一键提取助手

大家好！为了方便期末复习，我写了一个浏览器小插件，可以**一键把学习通的练习题提取成干净的纯文本**，方便大家直接复制到 Word 里自由排版和打印。

安装超级简单，支持各种电脑，跟着下面的步骤只需 1 分钟就能搞定！

## 🛠️ 第一步：安装浏览器插件（环境准备）

根据你平时用的浏览器，安装一个用来运行脚本的基础插件（只需安装一次）：

* **🪟 如果你用的是 Windows (Edge 或 Chrome 浏览器)：**
打开浏览器的“扩展/插件商店”，搜索并安装 **Tampermonkey**（认准那个黑色带有两个眼睛的“篡改猴”图标）。
* **🍎 如果你用的是 Mac (Safari 浏览器)：**
打开 Mac 自带的 App Store，搜索并下载 **Userscripts**。下载后，打开 Safari 顶部菜单栏的 `设置 -> 扩展`，把它打勾开启。

## 📦 第二步：一键安装提取脚本

基础插件安装好后，**直接点击下面这个链接**：

👉 **https://github.com/shenjunjie6666/chaoxing-exporter/raw/refs/heads/main/chaoxing.user.js**

* 点击后，你的浏览器会自动弹出一个脚本安装界面。
* 直接点击页面上的 **“安装” (Install)** 按钮，就大功告成了！

## ✨ 第三步：见证奇迹的时刻

1. 用浏览器登录学习通，点开任意一个你需要提取的**作业或练习题页面**（确保网页上已经显示了具体的题目）。
2. 留意网页的**右下角**，你会神奇地发现多出了一个绿色的 **“一键导出题目”** 悬浮按钮！
3. 点击它！几秒钟后，所有的题目和选项就会自动生成一个 `.txt` 文本文件下载到你的电脑里啦。

**💡 小贴士：**

* 下载的 `.txt` 文件可以直接用记事本打开，全选复制，粘贴到你的 Word 文档里随心排版。
* 如果打开学习通页面没有看到绿色按钮，按 `F5` 或 `Command+R` 刷新一下网页即可！
