# 🚀 学习通提取助手 (智能完美版 V3.2)

[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/)
[![Version](https://img.shields.io/badge/version-3.2-pro.svg?color=purple)](https://github.com/)

这是一款专门为期末复习打造的**非入侵式浏览器增强插件**。它可以一秒将网页上的练习题、作业转化为整洁的 Markdown 文本。**无需复杂的 Python 脚本，无需配置环境，1 分钟即可安装使用！**

> **💡 智能雷达技术**：本插件采用智能按需加载机制，平时不占用任何网页空间。只有当真正进入作业/练习题详情页时，尊贵的控制面板才会自动浮现，完美拒绝重影与卡顿！

---

## ✨ 四大核心黑科技

* **📋 一键复制 (Markdown)**：免去下载繁琐，点击直接复制整卷内容，粘贴至飞书、Notion 或 Word 即可获得完美排版。
* **📦 导出带答案版**：完整保留题干、选项、正确答案及超长解析，且解析内容采用高档灰底引用框包裹，一目了然。
* **📝 导出纯净版 (无答案)**：学霸突击刷题利器！一键隐去全部答案和解析，直接生成一张干净的“模拟自测卷”，方便打印重复练习。
* **❌ 智能错题本模式**：自动识别网页底部的得分和红叉 ❌，一键过滤全对题目，只把做错的题打包导出，精准复习。
* **🖼️ 突破图片限制**：完美兼容医学、理工科等带有图片的题目。自动修复“图片懒加载”陷阱，将图片转化为 Markdown 联网语法，在支持的编辑器中可直接显示图片！

## 🛠️ 安装教程 (保姆级)

### 第一步：准备运行环境
根据你平时使用的浏览器，安装对应的脚本管理器（只需安装一次）：
* **🪟 Windows (Edge / Chrome 等)**：在浏览器扩展商店搜索并安装 **Tampermonkey**（油猴）。
* **🍎 Mac (Safari 浏览器)**：在 App Store 搜索并下载 **Userscripts**，下载后在 Safari `设置 -> 扩展` 中勾选开启。

### 第二步：一键激活魔法
环境准备好后，**直接左键点击**下方链接即可触发自动安装页面（在弹出的油猴/Userscripts 界面点击“安装”或“确认”即可）：

👉 **[⚡️ 点击这里直接安装最新 V3.2 智能版]** https://github.com/shenjunjie6666/chaoxing-exporter/raw/refs/heads/main/chaoxing.user.js

## 📖 使用指南

1.  用浏览器登录学习通，点开任意一个你需要复习的**作业或练习题页面**。
2.  确保页面上已经显示出了具体的题目。此时，网页右下角会优雅地弹出一个**磨砂玻璃质感的控制面板**。
3.  根据你的复习需求，点击面板上对应的功能按钮即可！

## 🛠️ 技术声明 & 调试
* **绝对安全**：本插件属于纯前端 DOM 文本提取工具，只读取你当前屏幕上已有的文字，**不向服务器发送任何额外请求，不涉及自动刷课/作弊**，符合网络安全规范，请放心使用。
* **排版推荐**：导出的 `.md` 文件推荐使用 **Typora**、**Obsidian** 或 **Notion** 打开，可以获得完美的加粗标题、列表点和图片实时加载效果。
