# 📚 学习通练习题提取助手

一款轻量、高效的浏览器油猴（Tampermonkey）脚本。一键提取学习通（超星）测验、作业及考试回顾页面的题目及答案，支持快速导出为纯文本、Markdown 以及排版精美的 Word 文档，助你高效整理期末复习资料！

## ✨ 核心功能

* **🗂️ 多格式无缝导出**：支持直接复制纯文本，或一键下载为 TXT、Markdown、Word (`.doc`) 格式。
* **🎯 答案精准定位 (v3.6 新特性)**：智能识别网页结构，过滤“我的答案”，100% 精准提取官方“正确答案”。
* **🛡️ 强力防干扰提取**：采用“防贪婪匹配”算法，完美规避选项文本中包含英文单词（如 `EBV`、`DNA`）或特殊字符时导致的答案乱码。
* **🧹 智能排版净化**：自动清理题目自带的冗余编号（如 `1.`、`(1)` 等），导出后的题号永不重复、错乱。
* **🖱️ 无感悬浮面板**：右下角极简 UI 设计，防遮挡，不影响正常的网页浏览与页面操作。

## 🚀 安装指南

1. **安装脚本管理器**：如果你还没安装，请先在你的浏览器中安装 [Tampermonkey (油猴) 扩展](https://www.tampermonkey.net/)。
2. **安装本脚本**：
   * 打开本项目中的 `学习通提取助手.js` 文件。
   * 点击代码框右上角的 **Raw** 按钮，油猴会自动弹出安装界面，点击“安装”即可。

## 📖 使用说明

1. 登录学习通，进入任意包含题目的页面（如“作业详情”、“考试回顾”或“章节测验”）。
2. 等待页面加载完毕，脚本会自动检测题目。
3. 页面右下角会自动弹出 **“📚 提取助手”** 面板。
4. 点击你需要的导出格式（复制纯文本 / 导出 TXT / 导出 Markdown / 导出 Word），等待提示“✅ 导出成功”即可。
