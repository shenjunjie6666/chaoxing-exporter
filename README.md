# 📚 学习通 & 在浙学 练习题提取助手 v3.5

一个轻量、高效的浏览器油猴脚本（Tampermonkey），专为“学习通 (ChaoXing)”与“在浙学 (ZJOOC)”平台设计。一键提取测验、考试复盘页面的题目、选项、正确答案及解析，并支持导出为多种常用格式，方便复习与存档。

## ✨ 核心特性

* **🌐 双平台无缝支持**：自动识别当前所在网页，智能切换提取规则（支持 `chaoxing.com` 与 `zjooc.cn`）。
* **💾 多格式一键导出**：
* **复制纯文本**：直接写入剪贴板，随开随用。
* **TXT 导出**：纯净文本格式，适合快速预览。
* **Markdown 导出**：自带清晰排版（标题、引用、列表），适合笔记软件（如 Obsidian, Notion, Typora）。
* **Word 导出**：生成结构化的 `.doc` 文档，自动包含加粗标题、标准选项列表和醒目的正确答案标记，直接打印也美观。


* **🧠 智能文本净化**：
* 自动去除题目自带的重复序号（如 "1. "、"(3)" 等），由脚本统一重新编号，排版更整齐。
* 精准提取答案字母：无论原网页显示“正确答案: A”还是“✅ A”，脚本只提取纯净的字母（如 `A`、`ABC`），方便背诵和二次排版。


* **🖱️ 优雅的交互面板**：页面右下角悬浮操作面板，不遮挡做题视线，提供直观的操作反馈。

## 🛠️ 安装指南

1. **安装浏览器插件**：
首先需要在你的浏览器（Chrome, Edge, Firefox, Safari 等）上安装 [Tampermonkey (油猴)](https://www.google.com/search?q=https://www.tampermonkey.net/) 插件。
2. **安装本脚本**：
* **方式一**：在 Tampermonkey 控制面板中点击“添加新脚本”，清空默认代码，将本项目的脚本代码全选复制并粘贴进去，点击“文件” -> “保存”即可。
* **方式二**：点击 [这里](https://www.google.com/search?q=%23) 直接通过 GreasyFork 一键安装。



## 🚀 使用说明

1. 登录“学习通”或“在浙学”平台。
2. 进入具体的**作业、测验或考试的页面**（需要页面上已经显示出了题目）。
3. 稍等 1~2 秒，页面**右下角**会自动弹出一个带有 4 个按钮的控制面板。
4. 根据你的需求，点击对应的导出按钮：
* 点击 **“📋 复制纯文本”**，随后即可在任意地方 `Ctrl+V` 粘贴。
* 点击 **“💾 导出为 TXT”** 等按钮，浏览器会自动下载对应的文件，文件以 `平台名称题目_当前日期` 自动命名。
